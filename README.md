# Bot Manager

一个 [MCDReforged](https://github.com/Fallen-Breath/MCDReforged) 插件，需要 `2.3.0` 以上

### 命令

```
!!bot 显示本插件帮助信息
!!bot list 查看bot列表
!!bot add 添加bot（会自动获取玩家位置，将bot生成的位置设置到这）
!!bot remove 删除bot
!!bot modify 编辑你的bot名称或命令
!!bot run 运行bot的命令
!!bot reload 重载配置文件
!!bot info 查看bot详细信息
```

### 配置

```
"bots": 储存的bot列表
"bot_prefix": bot的前缀
"bot_suffix": bot的后缀
```
